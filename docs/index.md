# IChat Documentation

IChat is a Chrome side panel extension for context-aware AI chat on the web.

These pages are intended to be published from the repository `docs/` folder via GitHub Pages.

## Pages

- [Privacy Policy](./privacy-policy.md)
- [Extension Guide](./extension-guide.md)

## Product Summary

IChat helps users capture context from the current page and send it to a user-selected AI provider from a native Chrome side panel.

Current characteristics:

- Chrome MV3 extension
- local-first architecture
- BYOK provider setup
- no project-owned backend
- no built-in analytics or telemetry in the current implementation
