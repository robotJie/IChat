# IChat Logotype

This repository now includes a first-pass logotype system for `IChat` under [assets/brand](/E:/Projects/IChat/assets/brand).

## Deliverables

- `ichat-wordmark.svg`: primary monochrome wordmark
- `ichat-lockup-horizontal.svg`: symbol plus wordmark for README, banners, and wide headers
- `ichat-lockup-compact.svg`: stacked lockup for narrow placements
- `ichat-wordmark-small-mono.svg`: heavier mono version for 24-32 px usage
- `ichat-logotype-sheet.html`: one-page source for brand preview, usage rules, and context validation

## Brand decisions

- Keep `IChat` with an uppercase `I` to avoid Apple-like lowercase `i` cues.
- Use a rounded geometric stroke language so the mark feels calm and approachable rather than sci-fi.
- Build the companion symbol from the same `I` and `C` geometry instead of inheriting the current app icon.
- Use monochrome as the source of truth; apply the product's cold blue accent only as a secondary treatment.

## Suggested usage

- Use the horizontal lockup by default in README, documentation, and store-facing surfaces.
- Use the compact lockup for tight product placements that still need both symbol and name.
- Use the small mono wordmark in headers or labels where the horizontal lockup would crowd the layout.

## Validation

The brand sheet includes checks for:

- dark product header usage
- store or landing banner usage
- README or documentation hero usage
- clear space and minimum-size guidance
- common misuse examples
